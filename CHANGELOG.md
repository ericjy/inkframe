# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-03-03

### Added
- `inkframe render` — render images from markdown content with design or template
- `inkframe templates list` — list available templates
- `inkframe templates get` — get full template JSON or design-only
- `@file` prefix convention for passing file paths to `--content` and `--design`
- TypeScript SDK (`InkframeClient`) with `render()` and `listTemplates()` methods
